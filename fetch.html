Fetch

Using memory heuristics to change how we retain information.

Data generation is a distinctive mark of our age. If you are a knowledge worker, you may be dealing with this frustration daily. Windows of infinite tabs; folders accumulating a million “Untitled-01” files. This is because humans aren’t equipped to handle this much data. The sad part is, our operating systems/desktops haven’t evolved to handle such data either.

The goal of this project is to create a framework that unlocks the potential of how we store and retrieve our information in a humane way.
